## ✨ المميزات

### 🎭 الشخصية
- **الاسم:** نعنع 🌿
- **اللهجة:** أردنية أصيلة
- **الطابع:** طريف، ودود، ومفيد
- **الذكاء:** يفهم الأسئلة ويرد بشكل ذكي

### 💬 القدرات

#### 1. **الترحيب والتحية**
- يرد على السلام بطريقة طريفة
- يرحب بالزوار بحرارة
- يستخدم تعابير أردنية أصيلة

#### 2. **معلومات عن الباكجات**
يجيب على أسئلة مثل:
- "شو الباكجات عندكم؟"
- "شو في متوفر؟"
- "عندكم خدمات؟"

#### 3. **الأسعار**
يوضح الأسعار بطريقة واضحة:
- "الأسعار قديش؟"
- "كم السعر؟"
- "شو التكلفة؟"

#### 4. **كيفية الطلب**
يشرح خطوات الطلب بالتفصيل:
- "كيف أطلب؟"
- "شو الخطوات؟"
- "كيف أشتري؟"

#### 5. **العروض والخصومات**
يشرح نظام البذور (Seeds):
- "في عروض؟"
- "عندكم خصومات؟"
- "شو الكوبونات؟"

#### 6. **طرق الدفع**
يوضح طرق الدفع المتاحة:
- "كيف أدفع؟"
- "شو طرق الدفع؟"
- "في فيزا؟"

#### 7. **وقت التوصيل**
يجيب عن سرعة التوصيل:
- "كم بياخذ وقت؟"
- "متى بوصل؟"
- "سريع؟"

#### 8. **الأمان**
يطمئن العملاء:
- "آمن؟"
- "محمي؟"
- "بتطلبوا الباسورد؟"

#### 9. **الدعم الفني**
يوجه للتواصل:
- "كيف أتواصل؟"
- "عندي مشكلة"
- "بدي مساعدة"

## 🔧 التخصيص

### إضافة ردود جديدة:
افتح `chatbot.js` وعدّل دالة `processMessage()`:

```javascript
// مثال: إضافة رد على "شكراً"
if (this.matchKeywords(lowerMsg, ['شكرا', 'مشكور', 'يسلمو'])) {
    const responses = [
        "العفو حبيبي! 💚 أي خدمة!",
        "يسلمو! 😊 أنا هون دايماً!"
    ];
    this.addBotMessage(this.randomResponse(responses));
}
```

### تغيير الألوان:
افتح `chatbot.css` وعدّل المتغيرات:

```css
.nanaa-toggle {
    background: linear-gradient(135deg, #1E3D2F 0%, #C5A059 100%);
}
```
