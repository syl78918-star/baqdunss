## 🔧 التفاصيل التقنية

### **كيف يحسب النظام وقت النهاية:**
```javascript
const startDate = new Date(startTime);
const durationMs = parseInt(duration) * 60 * 1000;
const endTime = new Date(startDate.getTime() + durationMs);
```

**مثال:**
```
Start: 2026-02-14 15:00
Duration: 120 minutes

حساب:
startDate = Fri Feb 14 2026 15:00:00
durationMs = 120 × 60 × 1000 = 7,200,000 ms
endTime = startDate + 7,200,000 ms
       = Fri Feb 14 2026 17:00:00 ✅
```

### **كيف يشتغل التمديد:**
```javascript
const currentEnd = new Date(activeAuction.endTime);
const newEnd = new Date(currentEnd.getTime() + 5 * 60 * 1000);

activeAuction.endTime = newEnd.toISOString();
localStorage.setItem('baqdouns_active_auction', JSON.stringify(activeAuction));
```

**مثال:**
```
Current End: 17:00:00
Add: 5 × 60 × 1000 = 300,000 ms (5 دقائق)
New End: 17:05:00 ✅
```
