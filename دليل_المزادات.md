# 🔨 نظام المزادات المتقدم - دليل الاستخدام

## ✨ الميزات الجديدة

### 1️⃣ **مؤقت بدء ونهاية المزاد**
- الأدمن يحدد وقت البدء والنهاية
- عداد تنازلي live مرتبط بوقت النظام
- المزاد يبدأ ويتوقف تلقائياً

### 2️⃣ **تحويل تلقائي للفائز**
- لما ينتهي المزاد، الفائز يصير طلب في "طلباتي" تلقائياً
- الطلب من نوع "Auction Win"
- يظهر بشارة حمراء "AUCTION" في لوحة الأدمن

### 3️⃣ **نظام المراسلة**
- الأدمن يقدر يراسل الفائز مباشرة
- واجهة chat مريحة
- تحفظ كل الرسائل في الطلب

### 4️⃣ **نظام الإشعارات الشامل** 🔔 **جديد!**
- إشعارات تلقائية لجميع زوار الموقع
- يشتغل على كل الصفحات (index, game, seeds, etc.)
- 5 إشعارات في أوقات محددة:
  - ⏰ **30 دقيقة قبل البدء**: "مزاد قريب!"
  - 🔔 **5 دقائق قبل البدء**: "مزاد يبدأ قريباً جداً!"
  - 🚀 **عند البدء**: "المزاد بدأ الآن!"
  - ⚠️ **30 دقيقة قبل النهاية**: "المزاد ينتهي قريباً!"
  - 🔥 **5 دقائق قبل النهاية**: "المزاد ينتهي الآن!"
- 3 أنواع من الإشعارات:
  1. **Banner في أعلى الصفحة** (ملون ومتحرك)
  2. **Browser Notification** (إذا سمح المستخدم)
  3. **صوت تنبيه** (مختلف حسب الأهمية)

## 🚀 كيف تستخدم النظام (للأدمن)

### **الخطوة 1: تفعيل وضع الأدمن**
1. افتح صفحة المزادات: `auctions.html`
2. افتح console واكتب:
   ```javascript
   localStorage.setItem('baqdouns_admin_mode', 'true');
   location.reload();
   ```
3. سيظهر زر "🔧 Admin: Edit Auction"

### **الخطوة 2: ضبط المزاد**
1. اضغط على "🔧 Edit Auction"
2. املأ المعلومات:
   - **Title**: اسم المزاد (مثل: Premium Instagram Account)
   - **Handle**: username الحساب (@vip_jordan_99)
   - **Image URL**: صورة الحساب
   - **Followers**: عدد المتابعين (240k)
   - **Starting Bid**: السعر الابتدائي (500 بذرة)
   
3. **الجديد - نظام الوقت المبسّط:** 🆕
   - بدل ما تدخل وقت البدء ووقت النهاية منفصلين
   - الآن تدخل **وقت البدء + المدة**
   
   **مثال:**
   ```
   Auction Start Time: 2026-02-14 15:00
   Auction Duration (in minutes): 120
   ```
   - المدة بالدقائق (120 دقيقة = ساعتين)
   - النظام يحسب وقت النهاية تلقائياً!

4. **تأكيد:**
   سيظهر لك ملخص:
   ```
   ✅ Auction Updated!
   
   Start: 2026-02-14 15:00
   Duration: 120 minutes
   End: 2026-02-14 17:00
   ```

### **الخطوة 2.5: تمديد المزاد (جديد!)** 🆕
**أثناء سير المزاد**، الأدمن يقدر يمد الوقت:
- اضغط زر "⏰ Extend +5min" (أخضر)
- يضيف 5 دقائق تلقائياً لوقت النهاية
- مفيد لما يكون في مزايدات حامية!

**مثال:**
```
المزاد ينتهي الساعة 17:00
تضغط "Extend +5min"
→ الوقت الجديد: 17:05
```

💡 **ملاحظة**: يمكنك تمديد المزاد عدة مرات (كل مرة +5 دقائق)

### **الخطوة 3: مراقبة المزاد**
الصفحة ستعرض:
- ⏳ **قبل البدء**: "Starts in: 2h 30m 15s" (برتقالي)
- ⏰ **أثناء المزاد**: "Ends in: 45m 30s" (أحمر نابض)
- ✅ **بعد الانتهاء**: "Auction Ended" (أخضر)

### **الخطوة 4: انتهاء المزاد**
عند انتهاء الوقت **تلقائياً**:
1. المزاد يتوقف عن قبول مزايدات جديدة
2. يظهر alert بمعلومات الفائز
3. يتم إنشاء طلب جديد في `baqdouns_orders`
4. الطلب يحتوي على:
   - Order ID: `AUC-1707916800000`
   - Type: `Auction Win`
   - Winner Info: الاسم والإيميل
   - Amount: البذور المدفوعة
   - Status: `Pending Admin Contact`

### **الخطوة 5: التواصل مع الفائز**
1. افتح لوحة الأدمن: `anan.html`
2. اذهب إلى تبويب "Orders"
3. ابحث عن الطلب بشارة "AUCTION" الحمراء
4. اضغط على زر "💬 Message Winner" (أخضر)
5. تفتح نافذة المراسلة:
   - اكتب تفاصيل الحساب
   - اسم المستخدم وكلمة المرور
   - أي معلومات إضافية
6. اضغط "Send" أو Enter
7. الفائز سيشوف الرسالة في صفحة "طلباتي" عنده

## 🔧 تفاصيل تقنية

### **localStorage Keys:**
- `baqdouns_active_auction` - بيانات المزاد الحالي
- `baqdouns_auction_bid` - أعلى مزايدة
- `baqdouns_auction_bidder` - اسم المزايد
- `baqdouns_auction_bidder_email` - إيميل المزايد
- `baqdouns_auction_history` - تاريخ جميع المزايدات
- `baqdouns_auction_finished` - هل المزاد انتهى؟
- `baqdouns_orders` - جميع الطلبات (بما فيها المزادات)

### **مؤقت النظام:**
- يتحقق من الوقت كل ثانية (setInterval 1000ms)
- يربط بوقت النظام الحقيقي (`new Date()`)
- يحسب الفرق بين الوقت الحالي ووقت النهاية
- يعرض العداد التنازلي بتنسيق ديناميكي (أيام، ساعات، دقائق، ثواني)

### **إنشاء الطلب التلقائي:**
```javascript
const order = {
    id: 'AUC-' + Date.now(),
    type: 'Auction Win',
    userEmail: winnerEmail,
    userName: winnerName,
    auctionTitle: activeAuction.title,
    auctionHandle: activeAuction.handle,
    amount: winningBid,
    date: new Date().toISOString(),
    status: 'Pending Admin Contact',
    messages: []
};
```

### **نظام المراسلة:**
```javascript
// رسالة جديدة
{
    sender: 'Admin',  // أو اسم الفائز
    text: 'محتوى الرسالة',
    timestamp: '2026-02-14T18:05:30.000Z'
}
```

## ⚠️ ملاحظات مهمة

1. **استخدم الوقت الصحيح**: تأكد من كتابة التاريخ بالصيغة الصحيحة `YYYY-MM-DD HH:MM`
2. **مراعاة التوقيت**: الوقت المدخل هو بتوقيت المستخدم المحلي
3. **فحص البذور**: تأكد أن المزايد عنده بذور كافية قبل المزايدة
4. **استرجاع تلقائي**: المزايد السابق يسترجع بذوره فوراً عند مزايدة جديدة
5. **لا تغلق الصفحة**: اتركها مفتوحة حتى ينتهي المزاد بشكل صحيح

### **نظام الإشعارات:**
```javascript
// تتبع الإشعارات المُرسلة
baqdouns_sent_notifications: {
    '30min_before_start': true,
    '5min_before_start': true,
    'auction_started': true,
    '30min_before_end': true,
    '5min_before_end': true
}
```

**آلية العمل:**
1. يفحص حالة المزاد كل 30 ثانية
2. يحسب الفرق بين الوقت الحالي ووقت البدء/النهاية
3. عند الوصول للحظة محددة (مثلاً 30 دقيقة قبل البدء):
   - يرسل إشعار banner في أعلى الصفحة
   - يرسل browser notification (إذا كان مسموح)
   - يشغل صوت تنبيه
   - يحفظ أنه أرسل هذا الإشعار (لتجنب التكرار)

**ألوان الإشعارات:**
- 🟠 **تحذير** (warning): برتقالي - قبل البدء/النهاية بـ30 دقيقة
- 🔴 **عاجل** (urgent): أحمر - قبل البدء/النهاية بـ5 دقائق  
- 🟢 **نجاح** (success): أخضر - عند بدء المزاد

**للأدمن - إعادة ضبط الإشعارات (للتجربة):**
```javascript
// في console المتصفح
resetAuctionNotifications();
```
هذا يعيد تفعيل جميع الإشعارات من جديد (مفيد للتجربة)

### **2. تمديد المزاد الديناميكي** 🆕

**الفكرة:**
لما يكون المزاد حامي والناس تزايد بقوة، الأدمن يقدر يمد الوقت لإعطاء فرصة أكبر!

**كيف:**
1. اضغط زر "⏰ Extend +5min" (أخضر)
2. يضيف 5 دقائق فوراً
3. المزاد يستمر والعداد يتحدث

**أمثلة واقعية:**

**مثال 1: مزاد قريب ينتهي**
```
الوقت: 16:58
آخر مزايدة: 1500 بذرة (من أحمد)
16:59 - سارة تزايد 1600!
الأدمن: "في حركة! خلي نمد الوقت"
→ يضغط Extend +5min
الوقت الجديد: 17:04
17:01 - محمد يزايد 1700!
النتيجة: مزايدات أكثر = ربح أكثر! 💰
```

**مثال 2: تمديد متعدد**
```
المزاد ينتهي: 17:00
16:59 - Extend +5min → 17:05
17:04 - Extend +5min → 17:10
17:09 - Extend +5min → 17:15
✅ ممكن تمد قد ما تبي!
```

**متى تستخدم التمديد؟**
- 🔥 مزايدات حامية في آخر دقائق
- 👥 دخول مزايدين جدد
- 💎 حساب ثمين وتبي تزيد المنافسة
- ⏰ نسيت تحط وقت كافي من الأول

**فوائد:**
1. **زيادة الربح** - مزايدات أكثر = سعر أعلى
2. **مرونة** - تتحكم بالمزاد لحظياً
3. **عدالة** - كل الناس تاخذ فرصتها
4. **إثارة** - المزاد يصير أكثر تشويق!
